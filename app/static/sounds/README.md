# Sounds Directory

This directory contains audio files for the Kitchen Assistant AI.

## Required Files

### timer-finished.mp3
- **Purpose**: Plays when a timer finishes
- **Duration**: 1-2 seconds
- **Type**: Pleasant notification sound (e.g., bell chime, ding)
- **Volume**: Should be attention-grabbing but not jarring

## How to Add the Sound File

1. Find or create a short notification sound (MP3 format)
2. Name it `timer-finished.mp3`
3. Place it in this directory (`app/static/sounds/`)

## Free Sound Resources

You can find free notification sounds at:
- [Freesound.org](https://freesound.org/)
- [Mixkit.co](https://mixkit.co/free-sound-effects/)
- [Zapsplat.com](https://www.zapsplat.com/)

Search for: "bell", "notification", "chime", "ding", or "timer"
